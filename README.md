# 本地資源的深度整合

<!-- TODO 將該段介紹重新理的通順一些 -->

VS Code 的工作區域與檔案總管樹狀檢視雖然很方便，但當要與本地的其他資源互動時，比如想將下載的資源導入到專案中，或是想開啟圖片資料夾的參考檔案，只能跳脫 VS Code 的視窗，使用系統的檔案總管來操作，這樣就失去了在 VS Code 中一站式管理的便利性。

# 編輯體驗的全面進化

VS Code 原本的編輯體驗主要聚焦在程式碼檔案，對於圖片、多媒體等非程式碼資源的支援相對有限。本擴充套件強化了這些不足之處，提供更完整的檔案檢視、元資料顯示與操作整合，讓 VS Code 成為真正全能的工作環境。

## 圖片檢視器

取代 VS Code 預設的圖片編輯器，提供專業級的圖片檢視與互動體驗

| 功能名稱       | 功能描述                                                                                                  |
| -------------- | --------------------------------------------------------------------------------------------------------- |
| 完美骨架對齊   | 在 JavaScript 層重新實現 `object-fit: contain` 的邏輯，確保圖片載入前的骨架與實際圖片完美對齊，無閃爍跳動 |
| 手勢操作支援   | 支援滑鼠/觸控板縮放、平移、慣性滾動，在右鍵選單中可快速重設視角                                           |
| 吸管取色工具   | 內建 EyeDropper API 吸管工具，一鍵取色並自動複製色碼至剪貼簿                                              |
| 圖片格式轉換   | 右鍵選單提供「導出為...」選項，可將圖片轉換並儲存為其他格式（PNG、JPG、WebP 等）                          |
| 導出後快速檢視 | 導出完成後，通知中的「開啟檔案」按鈕可快速在編輯器中開啟新檔案，方便比較轉換前後的差異                    |

## 圖片剪貼簿

在 Windows 系統上提供進階的圖片複製功能，支援多種場景與格式

| 功能名稱       | 功能描述                                                                            |
| -------------- | ----------------------------------------------------------------------------------- |
| 檢視器內複製   | 在圖片檢視器的右鍵選單中可直接複製當前圖片                                          |
| 二進位資料複製 | 複製的是圖片二進位資料而非檔案路徑，可直接貼上至其他應用程式                        |
| 格式轉換支援   | 自動將 Windows 原生無法直接處理的格式（WebP、TIFF 等）轉換為 PNG 後複製，確保相容性 |
| 大型圖片進度   | 複製大型圖片時顯示即時進度條，提供清晰的視覺回饋                                    |

## 檔案元資料顯示

使 VS Code 能夠在管理非程式碼檔案的專案時也能輕鬆查看檔案的詳細資訊

| 功能名稱         | 功能描述                                                                                        |
| ---------------- | ----------------------------------------------------------------------------------------------- |
| 狀態列資訊顯示   | 自動在狀態列顯示當前開啟檔案的建立時間、修改時間與檔案大小                                      |
| 全檔案類型支援   | 支援所有檔案類型，包括 VS Code 原生無法開啟的任意二進位檔案（如 .blend、.psd 等）               |
| 圖片專屬資訊     | 針對圖片檔案提供特殊 UI，額外顯示解析度、長寬比、格式、色彩空間、色彩通道數、透明通道等詳細資訊 |
| Tab 切換自動更新 | 監聽 Tab 與 Tab Group 變化，自動更新狀態列顯示內容                                              |

## 自訂樣式注入

提供比 `Custom JS and CSS Loader` 更安全的方式來注入自訂 CSS 到 VS Code 介面

| 功能名稱        | 功能描述                                                                |
| --------------- | ----------------------------------------------------------------------- |
| 安全的 CSP 策略 | 採用修改而非刪除 Content-Security-Policy 的策略，維持安全完整性檢查     |
| 內聯 CSS 字串   | 在建構擴充套件時將 CSS 內聯為字串注入至 HTML，消除執行時的檔案 I/O 操作 |
| 命令式管理      | 提供命令來注入、還原與重新注入自訂樣式，方便管理與更新                  |
| 備份與還原      | 自動備份原始 workbench HTML 檔案，還原時可安全恢復                      |

## 外部應用程式整合

讓 VS Code 與系統原生應用程式無縫協作，實現快速的工作流程切換

| 功能名稱          | 功能描述                                                                                                  |
| ----------------- | --------------------------------------------------------------------------------------------------------- |
| 右鍵選單整合      | 在 VS Code 檔案總管右鍵任意檔案時，提供「使用系統預設應用程式開啟」選項                                   |
| Tab 右鍵支援      | 在編輯器 Tab 上右鍵也有相同選項，最佳化使用者體驗                                                         |
| 快速預覽 HTML/PDF | 可快速在預設瀏覽器中預覽 HTML、PDF 等檔案，無需離開 VS Code 視窗                                          |
| 專有格式支援      | 可直接開啟 .blend、.spp、.psd 等專有格式檔案在對應的原生應用程式中，讓你不需離開 VS Code 即可切換工作流程 |
