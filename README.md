# 1ureka 的個人 Visual Studio Code 擴充套件

本套件旨在強化 VS Code 對非程式碼資源的支援。透過新的系統級檔案瀏覽器、進階圖片元資料顯示及跨應用程式整合工具，優化從檔案管理到圖片檢視的完整工作流程，打造更直覺且安全的個人化開發環境。

---

# 編輯體驗的全面進化

VS Code 原本的編輯體驗主要聚焦在程式碼檔案，對於圖片、多媒體等非程式碼資源的支援相對有限。

本擴充套件強化了這些不足之處，提供更完整的檔案檢視、元資料顯示與操作整合，讓 VS Code 成為真正全能的工作環境。

## 圖片檢視器

該功能已分離為獨立的擴充套件 [1ureka 的 VS Code 預覽套件](https://github.com/1ureka/1ureka.vscode.preview)，專注於提供高品質的圖片檢視體驗，支援多種圖片格式與詳細的元資料顯示。

## 檔案元資料顯示

使 VS Code 能夠在管理非程式碼檔案的專案時也能輕鬆查看檔案的詳細資訊

| 功能名稱         | 功能描述                                                                                        |
| ---------------- | ----------------------------------------------------------------------------------------------- |
| 狀態列資訊顯示   | 自動在狀態列顯示當前開啟檔案的建立時間、修改時間與檔案大小                                      |
| 全檔案類型支援   | 支援所有檔案類型，包括 VS Code 原生無法開啟的任意二進位檔案（如 .blend、.psd 等）               |
| 圖片專屬資訊     | 針對圖片檔案提供特殊 UI，額外顯示解析度、長寬比、格式、色彩空間、色彩通道數、透明通道等詳細資訊 |
| Tab 切換自動更新 | 監聽 Tab 與 Tab Group 變化，自動更新狀態列顯示內容                                              |

## 自訂樣式注入

提供比 `Custom JS and CSS Loader` 更安全的方式來注入自訂 CSS 到 VS Code 介面

| 功能名稱        | 功能描述                                                                                            |
| --------------- | --------------------------------------------------------------------------------------------------- |
| 安全的 CSP 策略 | 採用修改而非刪除 Content-Security-Policy 的策略，只添加對於 Google Fonts 的例外，維持安全完整性檢查 |
| 命令式管理      | 提供命令來注入、還原與重新注入自訂樣式，方便管理與更新                                              |
| 內聯 CSS 字串   | 擴充套件注入時，會將 CSS 字串內聯在 workbench HTML，消除未來開啟應用時的檔案 I/O 操作               |
| 備份與還原      | 自動備份原始 workbench HTML 檔案，還原時可安全恢復                                                  |

## 外部應用程式整合

讓 VS Code 與系統原生應用程式無縫協作，實現快速的工作流程切換

| 功能名稱          | 功能描述                                                                                                  |
| ----------------- | --------------------------------------------------------------------------------------------------------- |
| 右鍵選單整合      | 在 VS Code 檔案總管右鍵任意檔案時，提供「使用系統預設應用程式開啟」選項                                   |
| Tab 右鍵支援      | 在編輯器 Tab 上右鍵也有相同選項，最佳化使用者體驗                                                         |
| 快速預覽 HTML/PDF | 可快速在預設瀏覽器中預覽 HTML、PDF 等檔案，無需離開 VS Code 視窗                                          |
| 專有格式支援      | 可直接開啟 .blend、.spp、.psd 等專有格式檔案在對應的原生應用程式中，讓你不需離開 VS Code 即可切換工作流程 |
