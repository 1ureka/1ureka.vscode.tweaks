# 1ureka's VSCode Extension

個人使用的 VSCode 擴展，整合多種實用功能。

## 主要功能

### 圖片檢視器
- 支援滑鼠/觸控板縮放、平移、慣性滾動
- 內建吸色器，一鍵取色並複製
- 右鍵選單快速重設視角
- 完整實作剪貼簿操作，避免無法刪除的 vscode 原生選單的選項無意義，使人誤導
- 取代 VsCode 預設圖片編輯器

### 圖片牆
- 提供三種布局模式（標準、編織、磚牆）和三種尺寸（小、中、大），可即時切換
- 預設的智慧瀑布流佈局，完美適應資料夾中大量不同尺寸的圖片
- 響應式設計，根據視窗大小自動適配 1-7 欄顯示
- 漸進式載入機制，以 SSR 精神即時顯示正確骨架排版，再透過訊息驅動逐張生成圖片，畫面完全不閃爍
- 自動壓縮圖片，即使資料夾中每張圖片為 50MB 以上的 PNG 也能高效顯示
- 與圖片檢視器一樣，實作剪貼簿操作，根據最後一次點擊位置判斷要複製的圖片路徑
- 右鍵資料夾「以圖片牆顯示」，快速瀏覽所有圖片
- 命令選擇區「開啟圖片牆」，快速瀏覽工作區外部的任意資料夾
- 點擊圖片即可在圖片檢視器中開啟

### 檔案元資料顯示
- 狀態列自動顯示當前檔案的建立、修改時間、大小等元資料
- 支援所有檔案類型，除了文字檔也包括無法開啟的二進位檔
- 可用於管理帶有非程式碼檔案的專案
- 針對圖片檔案有特殊UI，額外顯示解析度、長寬比、格式、色彩空間、通道等圖片資訊

### 自訂樣式注入
- 比起 `Custom JS and CSS Loader` 能更安全注入自訂 CSS 到 VSCode 介面
- 透過採用修改而非刪除 CSP 的策略，通過安全完整性檢查
- 以及建構插件時將 CSS 內聯為字串，消除執行時的檔案 I/O 也減少了安全風險

### 外部應用程式整合 (Windows)
- 右鍵 `.blend` 檔案快速以裝置中最新的 Blender 開啟
- 右鍵 `.spp` 檔案快速以裝置中最新的 Substance Painter 開啟
- 命令選擇區也可直接啟動 Substance Painter 或 Blender
