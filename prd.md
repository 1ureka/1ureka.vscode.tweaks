請完成該擴展插件專案，我已經幫你初始化好環境了，請你完成功能:

實現 esbuild.js 的 TODO，並先完成簡單的 hello world 功能，並且執行 npm run build 確認可以成功編譯
成功後刪除 hello world 功能，並實現以下功能:

在官方"檔案總管"中，右鍵以".blend"結尾的文件會出現 "以 Blender 開啟"功能
在 Crtl + Shift +P 中，出現 開啟 Blender 功能(直接開啟，不須選文件)

在官方"檔案總管"中，右鍵以".spp"結尾的文件會出現 "以 Painter 開啟"功能
在 Crtl + Shift +P 中，出現 開啟 Painter 功能(直接開啟，不須選文件)

以上四者在安裝時會檢查是否是windows來決定是否註冊，因此只需考慮windows

在開啟任意檔案時(包含文字編輯器支援的檔案或任意二進位檔案)，會在底下狀態欄右側顯示建立日期與修改日期，以compact的方式顯示，懸停顯示完整

在官方"檔案總管"中，右鍵任意資料夾時會出現 "以圖片牆顯示" 功能，並利用虛擬 URI 方式開啟自製編輯器UI，顯示圖片牆
在 Crtl + Shift +P 中，出現 開啟 圖片牆 功能，按下後會與官方的開啟資料夾功能一樣，出現該作業系統的檔案瀏覽器，可選擇資料夾
顯示邏輯是只顯示該資料夾的直接子項目，並且只顯示圖片檔案(jpg/png/bmp/gif/webp/tiff)，其他檔案忽略
